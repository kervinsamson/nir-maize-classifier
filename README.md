## Getting Started

### Jupyter Notebooks

1. **Initialize a Python virtual environment**

   ```bash
   python -m venv venv
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

### Frontend

Navigate to the frontend folder and start the development server:

```bash
cd app/frontend
npm run dev
```
